mICO

iOS Application that allows users to track Initial Coin Offerings and local cryptocurrency events

  • GitHub
  • GitHub

Description

mICO is an iOS Application that allows users to track Initial Coin Offerings and view local cryptocurrency events. This mobile platform will allow coin issuers and investors to focus more on the product and less on the details of the actual sale.

Technologies

React Native

Redux

Approach

  • Launched OAuth2.0 using passport, allowing users to authenticate using Google, Facebook, and LinkedIn
  • Implemented infinite scroll to automate axios requests to external API for real-time ICO data.
  • Integrated Google Maps API with geolocation to display nearby cryptocurrency events.
  • Recorded user activity using React Native Router Flux to create a seamless and responsive user experience during authentication process and app navigation.
